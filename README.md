# Guia do IF - IFSP-CJO

Um portal web informativo e intuitivo para alunos e comunidade do Instituto Federal de São Paulo - Campus Catanduva-José Olimpio (IFSP-CJO).

## 📋 Descrição

O **Guia do IF** é uma aplicação web que centraliza informações importantes do campus, facilitando o acesso rápido de novos alunos e da comunidade acadêmica a:

- 📅 Calendário acadêmico
- 👨‍🏫 Diretório de professores
- 📚 Informações da biblioteca
- ❓ Perguntas frequentes (FAQ)
- 📰 Últimas notícias oficiais
